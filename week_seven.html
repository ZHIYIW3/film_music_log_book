Making Music for Film and Animation 1 - Week 7 Log Book

Lecture Content and Reflection

MIDI

  • MIDI -> communication protocol that allows devices to send and receive specific data signals, able to interact with and play digital instruments and synthesizers
  • MIDI lets everyone easily write things on any instruments
  • Reaper:
    • Insert -> New Midi item -> record Midi elements
    • FX -> All Plugins -> List of instruments downloaded -> Add
    • Cancel Loop: Right-click track -> Item settings -> Untick ‘Loop item source’
    • Double click on Midi region -> Draw Notes -> Adjust the note length/height
    • View -> Virtual Midi keyboard -> Input 1 to VMK -> Play notes on keyboard

Project Tips

  • Watch the clip a few times, and identify all the main moments in the scene. What sound want to put in which moments?
  • Create markers for all the main moments at the start
  • Use audio automation to create fading in/out, make a soft transition
  • Can have multi-layer for one instrument, use different sound/effects

Red River (1948)

  • A classic American Western that uses an orchestral score
  • Has a romantic and sweet melody and harmony
  • The music creates a calm and peaceful atmosphere
  • All using a pretty slow tempo

High Noon (1952)

  • Use of songs in film - an element of the soundtrack and advertising of the film
  • The opening (highly known) track is repeating
  • More adventures, more rhythmic and paces, more fun
